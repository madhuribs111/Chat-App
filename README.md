## Run Locally

Clone the project

Go to project's root directory

Install dependencies

*In root directory:*

```npm install```

*Open another terminal window:*
``` bash
cd frontend
npm install
```
Start the server

```npm run server```

Start the client

*Open another terminal window*

```bash
cd frontend
npm run dev
```
